<h1>Calculadora</h1>
<h2>Mini calculadora feita em java</h2>
<p>Foi utilizado nesse projeto  programação orientada a objetos, encapsulamentos e alguns calculos básicos, como: Soma, subtração, divisão e multiplicação</p>
